HOW To Describe Images

This section is broken into two distinct segments. The first, "General Guidelines," details best practices that apply when describing any image. This includes consideration of key elements such as context, audience, layout, function, and tone. If you've ever wondered about the appropriate length of text or inquired about how you should address color of your description, be sure to read through these guidelines!

The second segment, "Specific Guidelines," presents best practice guidelines for particular image categories and classifications. For example, the recommended approach for writing a description for a bar graph will be very different when addressing a map. This section aims to present the nuances specific to types of images to help content producers write effective descriptions for their target audience.
