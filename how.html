HOW To Describe Images

This section is broken into two distinct segments. The first section, "General Guidelines," details best practices that apply when writing a description for any image. This includes consideration of key elements such as context, audience, layout, function, and tone. If you've ever wondered about the appropriate length of text or inquired about how you should address color in your description, be sure to review this section!

The second segment of this page details best practice guidelines specific to particular image categories and classifications. For example, the recommended approach for writing a description for a bar graph will differ significantly from the approach applied to a map or flow chart. This section aims to present nuances particular to specific types of images in efforts to help content producers write effective descriptions for their target audience.
